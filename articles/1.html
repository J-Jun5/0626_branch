Origin과 master, HEAD

Git은 "원격저장소", "로컬저장소", "작업 폴더" 이렇게 세가지 장소가 있습니다.

1. origin
: 원격 저장소를 의미 함.

2. master
: git에서 사용되는 기본 branch 이름. (*main과 master의 차이점은?
: 둘 다 같은 기본 branch이지만 GitHub를 비롯한 여러 플랫폼에서 기본 브랜치의 이름을 'main'으로 변경함. 포용성과 다양성을 고려한 결과.)

3. HEAD
: 현재 어떤 작업 공간에 있는지 나타냄.
현재 checkout된 commit 또는 branch를 가리키는 포인터. "origin/HEAD"를 사용해 원격 저장소의 현재 작업 위치를 알 수 있음.
