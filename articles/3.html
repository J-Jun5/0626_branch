branch

clone과 pull, fetch

1. git clone
: 'git clone' 커멘드는 remote 저장소를 복제하여 로컬에 새로운 저장소를 생성한다.
remote 저장소의 모든 commit 기록과 브랜치를 포함한 전체를 로컬에 복제한다

2. git pull
: 'git pull' 커멘드는 remote 저장소의 변경 사항을 로컬 저장소로 가져와 현재 branch에 병합한다.
'git fetch'와 'git merge'를 결합한 명령어

3. git fetch
: 'git fetch' 커멘드는 remote 저장소의 변경 사항을 로컬 저장소로 가져오지만,
현재 branch에 병합하지는 않는다. remote의 최신 상태를 로컬 장소에 업데이트 하는데 사용함
